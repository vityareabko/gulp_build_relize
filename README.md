# gulp_build_relize
2 основные команды
1) gulp watch - следить за изменениями файлов scss js css html, и обновлять страницу в браузере
2) gulp - собирает проект, сжимает изображения и конкатинирует файли css, js + переносит в папку dist

команды
gulp cleans
gulp styleSCSS
gulp img-compresed
gulp listen_html
